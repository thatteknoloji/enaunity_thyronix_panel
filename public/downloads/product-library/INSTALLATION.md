# ENA Marketplace Connector

Kurulum:

1. `Chrome`, `Edge`, `Brave` veya `Opera` acin.
2. `chrome://extensions` adresine gidin.
3. `Developer mode / Gelistirici modu` acin.
4. `Load unpacked / Paketlenmemis yukle` ile `public/product-library/connector` klasorunu secin.

Alternatif:

- `ena-marketplace-connector.zip` dosyasini indirip klasore cikarin.
- Sonra cikardiginiz klasoru `Load unpacked` ile secin.

Kullanim:

1. ENA panelinde `Bayi > Product Library > Magaza Kuyrugu` ekranina gidin.
2. Paket recetesinden `Magazaya Gonder` ile is olusturun.
3. Connector popup'inda magazayi secin.
4. `Simdi Calistir` veya `Otomatik Calis` ile kuyrugu isletin.

Notlar:

- Ilk surum `Chromium` ailesi icindir.
- Pazaryeri panelinde oturumun acik olmasi gerekir.
- Import sayfasi cok farkli degismisse connector dosya inputunu bulamayabilir; bu durumda pazaryeri import ekranini manuel acin ve tekrar deneyin.
